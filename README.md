# Level 2: Application

## Install and run

You need install:

- [Node.js](https://nodejs.org) and [Npm](https://www.npmjs.com/)

Write in command line

```bash
git clone https://github.com/aleksnick/funbox-test.git
cd funbox-test
npm install
npm run start
```

If you encounter problems, run:

```bash
npm run build
```

and open `index.html` in your browser

## Run tests

Write in command line

```bash
npm run test
```
